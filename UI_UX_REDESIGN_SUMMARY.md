```
Primary:        #3b82f6 (Bright Blue)

```

**Why this palette?**
- ✓ High contrast for accessibility
- ✓ Modern, tech-forward aesthetic
- ✓ Professional appearance
- ✓ Reduces eye strain (dark theme)
- ✓ Creates visual hierarchy with gradient accents

### Typography System
- **Display Font:** Poppins (headings) — Bold, modern
- **Body Font:** Inter (content) — Clean, readable
- **Hierarchy:** 6 heading levels with proper sizing
- **Line Heights:** Optimized for readability (1.2 - 1.75)
- **Letter Spacing:** Professional tracking on titles

### Spacing & Layout
- **8px Base Grid** for consistent alignment
- **Spacing Scale:** 4px to 80px increments
- **Border Radius:** Smooth, modern corners
- **Shadows:** Layered depth for premium feel
